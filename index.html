Web2Char

With this tool you can create a *.chumpy file that stores your character. This file is basically a JSON document, that will be the heart of your character. It is compatible with all the tools from Chumpy. For now you can just use it to store your character, but add e.g. git for version control and you already got yourself a tool to manage character development. You can of course also hack your own tools to view, edit and modify your Chumpy character ! Stay tuned for more tools - this is just the most fundamental basis, more will come in the future !
